# Badges for README

Add these badges to your README.md to show the project status:

```markdown
[![CI](https://github.com/yourusername/denv/actions/workflows/ci.yml/badge.svg)](https://github.com/yourusername/denv/actions/workflows/ci.yml)
[![Release](https://github.com/yourusername/denv/actions/workflows/release.yml/badge.svg)](https://github.com/yourusername/denv/actions/workflows/release.yml)
[![Go Report Card](https://goreportcard.com/badge/github.com/yourusername/denv)](https://goreportcard.com/report/github.com/yourusername/denv)
[![codecov](https://codecov.io/gh/yourusername/denv/branch/main/graph/badge.svg)](https://codecov.io/gh/yourusername/denv)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Go Version](https://img.shields.io/github/go-mod/go-version/yourusername/denv)](https://go.dev/)
[![Latest Release](https://img.shields.io/github/v/release/yourusername/denv)](https://github.com/yourusername/denv/releases/latest)
[![Downloads](https://img.shields.io/github/downloads/yourusername/denv/total)](https://github.com/yourusername/denv/releases)
[![Docker Pulls](https://img.shields.io/docker/pulls/yourusername/denv)](https://hub.docker.com/r/yourusername/denv)
```

## Usage

Copy and paste this at the top of your README.md, right after the title:

```markdown
# denv

[![CI](https://github.com/yourusername/denv/actions/workflows/ci.yml/badge.svg)](https://github.com/yourusername/denv/actions/workflows/ci.yml)
[![codecov](https://codecov.io/gh/yourusername/denv/branch/main/graph/badge.svg)](https://codecov.io/gh/yourusername/denv)
[![Go Report Card](https://goreportcard.com/badge/github.com/yourusername/denv)](https://goreportcard.com/report/github.com/yourusername/denv)
[![Latest Release](https://img.shields.io/github/v/release/yourusername/denv)](https://github.com/yourusername/denv/releases/latest)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Zero-configuration development environment manager
```

Remember to replace `yourusername` with your actual GitHub username!
